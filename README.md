# Keras_Apartment_Rent
t.Kerasで線形回帰モデルを構築し、アパートの家賃を予測する。

## Description
 
大まかなコードの流れ
- データセットの作成
- 線形回帰モデルを構築
- モデルコンパイル
- モデル学習
- アパートの家賃を予測
